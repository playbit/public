# Playbit public feedback

Please [file issues](https://github.com/playbit/public/issues) with care and love


## Reporting bugs & providing feedback

If an application crashes, a pop-up will give the option to "Debug" it. This will open the core dump in the debugger. Take a screenshot that includes the the information displayed in the core as well as the version number in the bottom-right corner of the screen. Then, send this screenshot to Playbit, along with a description of the steps leading up to the crash.

If the _workspace manager crashes_, the desktop app will unexpectedly quit.
Please attach all the files at `~/Library/Logs/Playbit*` in your issue
with a description of the steps leading up to the crash.

If the _desktop app crashes_, macOS will generate a crash report.
Please attach this report in an issue along with all files at `~/Library/Logs/Playbit*`,
with a description of the steps leading up to the crash.
